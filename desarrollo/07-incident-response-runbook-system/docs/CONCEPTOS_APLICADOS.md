# Conceptos aplicados

## Incident response

El dominio modela severidad, estado y servicio afectado para hablar de operacion real.

## SLI/SLO

El siguiente paso es asociar incidentes a SLOs afectados y calcular impacto.

## Blameless postmortem

El proyecto debe evolucionar agregando una entidad `Postmortem` con causa raiz, impacto y acciones correctivas.
