# Roadmap 2026-2027

## Objetivo

Llegar a enero-marzo de 2027 con opciones reales a roles internacionales Senior Backend / Cloud / Lead Java, idealmente en el rango de **80K-100K EUR total compensation** o superior segun empresa, alcance y paquete.

## Mayo 2026: ordenar perfil

### Foco

- Reescribir CV en ingles.
- Actualizar LinkedIn.
- Definir posicionamiento unico.
- Preparar pitch profesional.
- Identificar brechas reales.

### Entregables

- CV v1.
- LinkedIn actualizado.
- Pitch de 90 segundos.
- Lista de 50 empresas objetivo.
- Tabla de evidencias reales por tecnologia.

## Junio 2026: Java moderno y backend base

### Foco

- Java 17/21.
- Spring Boot 3.
- APIs REST.
- PostgreSQL.
- Flyway.
- JPA con criterio.
- JUnit 5.
- Testcontainers.
- Docker Compose.
- GitHub Actions.

### Entregable

Primer corte del proyecto tecnico principal:

- customer-service,
- ticket-service,
- PostgreSQL,
- OpenAPI,
- tests de integracion,
- pipeline CI.

## Julio 2026: observabilidad y resiliencia

### Foco

- Micrometer.
- Actuator.
- OpenTelemetry.
- Logs estructurados.
- Correlation ID.
- Prometheus/Grafana o CloudWatch.
- Resilience4j.
- Timeouts.
- Retries.
- Circuit breakers.

### Entregables

- Dashboard basico.
- Trazas entre servicios.
- Logs correlacionados.
- Runbook simple de incidentes.
- 4 ejercicios de system design.

## Agosto 2026: AWS y eventos

### Foco

- AWS Developer Associate como certificacion objetivo.
- ECS/Fargate.
- RDS.
- SQS/SNS.
- CloudWatch.
- IAM.
- Event-driven architecture.
- DLQ.
- Idempotencia.
- Outbox pattern.

### Entregables

- Flujo asincrono: ticket creado -> evento -> notificacion -> auditoria.
- Consumidores idempotentes.
- DLQ.
- Estimacion de costos AWS.
- Proyecto documentado con README y diagrama simple.

## Septiembre 2026: mercado

### Foco

- Empezar aplicaciones reales.
- Medir respuesta.
- Practicar recruiter calls.
- Ajustar CV por feedback.

### Actividad minima

- 30-40 aplicaciones.
- 10-15 contactos recruiters.
- 5 mensajes a hiring managers.
- 2 referrals buscados.
- 10 problemas coding.
- 2 system designs.

## Octubre 2026: system design senior

### Foco

- Arquitectura de pagos.
- Auditoria.
- Sistemas de notificacion.
- Rate limiting distribuido.
- Monitoring platform.
- Search/autocomplete.
- Trade-offs.

### Entregables

- 8 system designs documentados.
- 2 mocks grabados.
- 10 historias STAR refinadas.
- CV v2.

## Noviembre 2026: IA aplicada como complemento

### Foco

Solo si no interfiere con mercado y entrevistas.

- LLM APIs.
- Structured outputs.
- Prompt versioning.
- Rate limits.
- Cost control.
- Fallback provider.
- Logging seguro.
- Evaluacion basica.

### Entregable opcional

Mini proyecto:

**ai-support-assistant**

Funciones:

- clasificar tickets,
- extraer entidades,
- resumir casos,
- sugerir respuesta,
- devolver JSON estructurado,
- medir latencia y costo.

## Diciembre 2026: segunda ola

### Foco

- Reforzar entrevistas falladas.
- Mejorar ingles tecnico.
- Cerrar gaps de AWS.
- Preparar negociacion.
- Enviar segunda ola de aplicaciones.

### Entregables

- CV v3.
- LinkedIn v2.
- 20 historias STAR.
- 30 nuevas aplicaciones.
- 5 referrals.
- AWS Developer Associate programada o aprobada.

## Enero 2027: cierre

### Foco

- Reactivar procesos.
- Priorizar empresas Tier A y B.
- Mantener pipeline abierto.
- Negociar sin anclar bajo.

### Criterios para aceptar oferta

- 80K EUR TC o mas, idealmente.
- Rol senior real.
- Stack moderno.
- Exposicion internacional.
- Marca profesional mejor que la actual.
- Crecimiento hacia Backend Architect, Senior Cloud Engineer o Applied AI Backend.
- On-call razonable.
- Cultura tecnica sana.
